Trabajar con Strings

Propiedades

.length

.length nos dice cuantos caracteres tiene ese texto:

                let a = 'JavaScript'
                console.log(a.length)
                <- 10

Métodos sin parametros

.trim()

.trim() elimina los espacios en blanco en un string

                let a = '     JavaScript     '
                console.log(a.trim())
                <- "JavaScript"

.toUpperCase()

.toUpperCase() convierte todo el string a mayúsculas

                let a = 'javascript'
                console.log(a.toUpperCase())
                <- "JAVASCRIPT"

.toLowerCase()

.toLowerCase() convierte todo el string a minúsculas

                let a = 'JAVASCRIPT'
                console.log(a.toLowerCase())
                <- "javascript"
