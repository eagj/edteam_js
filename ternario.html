operador ternario

El operador ternario sirve para hacer una comprobación de true false en una sola linea:

                    let edad = prompt('Cual es tu edad')
        
                    let adulto = edad >= 18
                        ? 'Eres mayor de edad'
                        : 'Eres menor de edad'
                    
                    alert('Tienes ' + edad + ' años. ' + adulto)
